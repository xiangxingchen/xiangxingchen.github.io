JS 异常处理

对于 Javascript 而言，我们面对的仅仅只是异常，异常的出现不会直接导致 JS 引擎崩溃，最多只会使当前执行的任务终止。

  • 当前代码块将作为一个任务压入任务队列中，JS 线程会不断地从任务队列中提取任务执行。
  • 当任务执行过程中出现异常，且异常没有捕获处理，则会一直沿着调用栈一层层向外抛出，最终终止当前任务的执行。
  • JS 线程会继续从任务队列中提取下一个任务继续执行。
